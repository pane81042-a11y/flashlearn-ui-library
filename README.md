# FlashLearn UI Library

FlashLearn UI Library is a shared component system used across all FlashLearn microfrontends.

It ensures consistent design and reusable UI components.

## Purpose

- Maintain visual consistency
- Reduce duplication
- Centralize UI updates
- Improve scalability

## Components

- Buttons
- Inputs and Forms
- Study Cards
- Layout components
- Modals
- Notifications
- Typography system
- Loading states

## Architecture

Built with:

- React
- TypeScript (recommended)
- CSS Modules or Styled Components

Used in:

- flashlearn-shell-frontend
- flashlearn-deck-frontend
- flashlearn-card-frontend
- flashlearn-study-frontend
- flashlearn-progress-frontend
- flashlearn-auth-frontend
- flashlearn-admin-frontend

## Distribution

- Private npm or GitHub Packages
- Version controlled releases

## Portfolio Value

Demonstrates:

- Design system architecture
- Reusable component engineering
- Scalable microfrontend UI structure
